# 協作型生成式 AI 代理人程式考試自動評分方法

本專案實作一套基於 **生成式 AI 與多代理人協作機制** 的程式考試自動評分方法，
透過模擬人類批改流程中的「獨立判斷、意見比較與仲裁決策」，以提升評分結果在
一致性、可解釋性與穩定性上的表現。

本 Repository 聚焦於 **研究方法本身與批改行為分析**，並非產品化系統或教學平台。

---

## 📌 方法概述

本方法將自動程式評分流程拆解為多個具備不同角色的 AI 代理人，
並透過協作與仲裁機制整合評分結果，核心設計包含：

- 多代理人獨立批改，降低單一模型偏誤
- 評分結果之語意比對與共識判定
- 評分分歧時引入仲裁代理進行最終決策
- 於評分前加入輸入內容檢查，避免語意操控影響批改行為

---

## 🧠 系統流程概念

整體流程可概略分為兩個階段：

### 1️⃣ 輸入檢查階段
- 檢視學生作答是否包含具語意操控風險的內容
- 異常輸入不進入後續批改流程

### 2️⃣ 協作評分階段
- 兩位批改代理人依相同批改題詞進行 **獨立評分**
- 系統比較雙方評分結果與批改理由
- 若結果一致，直接輸出評分
- 若出現分歧，則由仲裁代理人介入整合判斷

---

## 🧪 實驗設計說明

本專案僅包含以下兩類批改實驗結果：

### 🔹 不同作答水準之批改結果
- 高分作答
- 中等分作答
- 低分作答  

用以觀察多代理人在不同品質輸入下的評分行為差異。

### 🔹 隨機批改順序實驗
- 對相同作答進行多次、隨機順序的批改
- 分析評分結果在不同批改順序下的穩定性

所有結果皆用於 **分析代理人決策行為與評分一致性**，  
不涉及實際課程成績或教學評量。
